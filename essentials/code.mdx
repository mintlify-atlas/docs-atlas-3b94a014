## Inline code

To denote a `word` or `phrase` as code, enclose it in backticks (`).


## Code blocks

Use [fenced code blocks](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) by enclosing code in three backticks and follow the leading ticks with the programming language of your snippet to get syntax highlighting.

```java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

````md
```java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
````

## Code block with title

Optionally, you can add a title to your code block by adding it after the programming language.

```java HelloWorld.java

    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

````md

````

## Syntax highlighting

Mintlify supports syntax highlighting for all major programming languages. Simply specify the language after the opening backticks.

<Tabs>
  <Tab title="JavaScript">
    ```javascript
    function greet(name) {
      return `Hello, ${name}!`;
    }
    ```
  </Tab>
  <Tab title="Python">
    ```python
    def greet(name):
        return f"Hello, {name}!"
    ```
  </Tab>
  <Tab title="TypeScript">
    ```typescript
    function greet(name: string): string {
      return `Hello, ${name}!`;
    }
    ```
  </Tab>
</Tabs>

## Advanced code block features

### Line highlighting

Highlight specific lines in your code block by adding `{line-numbers}` after the language.

```javascript {2,4-6}
function calculateTotal(items) {
  const subtotal = items.reduce((sum, item) => sum + item.price, 0);
  const tax = subtotal * 0.1;
  const shipping = 5.99;
  const discount = subtotal > 100 ? 10 : 0;
  return subtotal + tax + shipping - discount;
}
```

### Diff markers

Show additions and deletions in code using `+` and `-` at the start of lines.

```javascript
function greet(name) {
-  return "Hello " + name;
+  return `Hello, ${name}!`;
}
```

````md
```javascript

-  return "Hello " + name;
+  return `Hello, ${name}!`;
}
```
````

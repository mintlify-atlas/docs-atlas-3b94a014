## Images

### Using Markdown syntax

The [markdown syntax](https://www.markdownguide.org/basic-syntax/#images) lets you add images using the following code:

```md
![title](/path/to/image.jpg)
```

<Note>
Image file sizes must be less than 5MB. For larger images, host them on a service like [Cloudinary](https://cloudinary.com/) or [S3](https://aws.amazon.com/s3/) and use the URL.
</Note>

### Using HTML img tags

To get more customizability with images, you can use HTML `<img>` tags with inline styles:

```html
<img height="200" src="/path/to/image.jpg" alt="Description" />
```

Example with styling:

```html
<img
  style={{ borderRadius: '0.5rem', maxWidth: '600px' }}
  src="/images/screenshot.png"
  alt="Application screenshot"
/>
```

## Video embeds

Embed videos using iframes. Here's an example with YouTube:

<iframe
  width="560"
  height="315"
  src="https://www.youtube.com/embed/4KzFe50RQkQ"
  title="YouTube video player"
  frameBorder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowFullScreen
  style={{ width: '100%', borderRadius: '0.5rem' }}
></iframe>

## HTML elements

<Tip>
Mintlify supports [HTML tags in Markdown](https://www.markdownguide.org/basic-syntax/#html). This is helpful if you prefer HTML tags to Markdown syntax, and lets you create documentation with infinite flexibility.
</Tip>

You can use any HTML element in your MDX files:

- `<iframe>` - Embed external content like videos
- `<img>` - Display images with custom styling
- `<video>` - Embed video files directly
- `<a>` - Create links with custom attributes
- `<div>`, `<span>` - Structure and style content

### Example: Custom container

```html
<div style={{ 
  padding: '20px', 
  background: '#f5f5f5', 
  borderRadius: '8px',
  marginTop: '16px'
}}>
  <h4>Custom Container</h4>
  <p>You can create custom styled containers using HTML and inline styles.</p>
</div>
```
